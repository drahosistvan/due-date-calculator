# Due Date Calculator

## Install
1. Clone the repository `git clone https://developer.isti.cloud/drahos.istvan/due-date-calculator.git`
2. Install composer dependencies `composer install`
3. Add proper rights to the application `chmod +x ./application`

## Application run
`./application duedate:calculate "2017-10-23 15:30:00" 2`

__Argument 1:__ Submit day (format: YYYY-MM-DD HH:MM:SS)

__Arugment 2:__ Turnaround time (in hours)
